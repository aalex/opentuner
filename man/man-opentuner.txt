[DESCRIPTION]
Opentuner is a musical tuner.

[SYNOPSIS]
opentuner [options]

[EXAMPLES]

To start it:

 opentuner

[INTERACTIVE USAGE]

Opentuner is an interactive software. While it runs, the user can control it using keyboard keys.
